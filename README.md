# Copfinder
## Структура
- `src/cli` — точки входа CLI.
- `src/config` — загрузка конфигов и переменных окружения.
- `src/copart` — логика браузерного парсинга Copart.
- `src/iaai` — HTML-парсер IAAI и крон-планировщик.
- `src/services` и `src/utils` — переиспользуемые сервисы/утилиты.
- `config` — декларативные JSON-конфиги + пример env-файла.
- `data` — состояние (история отправленных лотов и т.п.).
- `artifacts` — тяжёлые HTML-дампы.
- `aws/` — конфигурации и скрипты для развертывания на AWS.
